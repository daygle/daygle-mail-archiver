# Daygle Mail Archiver Wiki

Welcome to the Daygle Mail Archiver documentation! This wiki provides comprehensive guides for installing, configuring, and using the Daygle Mail Archiver.

## Overview

Daygle Mail Archiver is a deterministic email ingestion and archiving system designed for longâ€‘term retention, auditability, and operational reliability. It ingests emails from multiple sources (IMAP, Gmail API, Office 365 Graph API), stores messages in a structured database, and exposes a clean UI for browsing, retention policy management, and administrative control.

## Key Features

- **Multi-Source Email Fetching**: IMAP/IMAPS, Gmail API, Office 365 Graph API
- **Automatic Email Archiving**: Continuously polls accounts and stores emails
- **Search & Filter**: Full-text search across subjects, senders, and recipients
- **Retention Policies**: Automatic purging based on configurable rules
- **User Management**: Multi-user system with role-based access
- **Dashboard Analytics**: Visual charts showing email statistics
- **Virus Scanning**: Integrated ClamAV for email security
- **Database Backup & Restore**: Built-in backup functionality
