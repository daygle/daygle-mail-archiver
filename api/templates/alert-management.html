How Alerts Work
Alert Generation Logic

An alert is only created and sent when:

  1. Trigger is enabled (configured above)
  2. Severity level is enabled globally (configured above)
  3. User has email notifications enabled (in user settings)
  4. User has the severity level enabled (in user settings)
Alert Hierarchy
  • Global Settings: Alert types (error/warning/info/success)
  • Alert Triggers: Specific events (virus detected, etc.)
  • User Preferences: Per-user alert type preferences
  • Email Settings: SMTP configuration for delivery
